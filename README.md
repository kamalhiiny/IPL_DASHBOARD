# IPL Dashboard

## Overview
A simple static dashboard built from `advanced_ipl_dataset_500.csv` and rendered via `ipl_dashboard.html`.

## Files
- `ipl_dashboard.html`: main dashboard page
- `advanced_ipl_dataset_500.csv`: IPL dataset source

## Run locally
1. Open a terminal in `c:\Users\SRAVYA\Downloads\srmvdp`
2. Start server:
   - `python -m http.server 8000`
3. Visit in browser:
   - `http://localhost:8000/ipl_dashboard.html`
